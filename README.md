## Installation

To set up the project, follow these steps on command line:

1. Clone the repository:

   ```bash
   git clone https://github.com/lasttang0/shooter-app.git
   cd shooter-app
   ```

2. Install the dependencies:

   ```bash
   pip install -r requirements.txt
   ```
   
## Launch

Launch the game:

   ```bash
   python main.py
   ```
